# Pixelando 👾

Tienda online de hama beads artesanales con estilo pixel art medieval.

## Descripción

Pixelando es una tienda donde se pueden ver y encargar creaciones únicas de hama beads hechas a mano. El diseño está inspirado en la estética de taberna medieval pixel art, con animaciones y elementos visuales temáticos.

## Características

- Página de inicio con animaciones (Framer Motion) y cofre interactivo
- Banner animado con scroll infinito
- Tabla de rarezas (Común, Raro, Épico, Legendario, Mítico) con precios
- Selector de tipo de cuenta: Estándar (5mm) y Detalle Premium (2,6mm)
- Galería de creaciones reales con grid responsive
- Formulario de encargos con rareza preseleccionada y subida de foto
- Página de acceso con login y registro
- Validaciones de formulario con requisitos de contraseña en tiempo real
- Navegación entre páginas con React Router
- Diseño responsive con menú hamburguesa
- Footer con información de contacto y redes sociales

## Stack

- React + Vite
- Framer Motion
- React Router DOM
- React Icons
- CSS modular por componente

## Requisitos

- Node.js instalado
- npm instalado

## Instalación

```bash
npm install
npm run dev
```
